NetApp SaaS Backup for Microsoft 365 是一款基于 Web 的安全软件即服务（ Software-as-a-Service ， SaaS ）产品，可将 Microsoft 365 数据备份到 Amazon S3 存储或 Microsoft Azure Blob 存储。SaaS Backup 为空闲和传输中的数据提供加密。
