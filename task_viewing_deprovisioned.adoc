= 查看已取消配置的项目列表
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以查看已取消配置的邮箱或用户帐户列表。

.步骤
. 单击左侧导航窗格中的 * 服务 * 。
. 在所需的服务中，单击未受保护的项数。
. 单击 * 已取消配置 * 选项卡。
